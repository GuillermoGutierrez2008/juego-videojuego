# Carpetas de imágenes

Pegá acá tus imágenes:

- `campeones/` — splash arts y avatares de campeones (yasuo.jpg, ahri.png, etc.)
- `logos/`     — logos de LoL, Riot, etc.
- `fondos/`    — imágenes de fondo (Grieta del Invocador, etc.)

Las páginas funcionan sin imágenes (usan placeholders con la inicial del nombre),
pero podés reemplazarlos en el HTML por `<img src="img/campeones/yasuo.jpg">`.
